# Portfolio Site

A simple starter project for a personal portfolio website.

## Setup

1. Open `index.html` in a web browser to view the site.
2. Add your custom styles to `style.css`.
3. Add interactive behavior in `script.js`.
4. Place images and other static assets in the `assets/` directory.

## Structure

- `index.html` – Main HTML file.
- `style.css` – Stylesheet.
- `script.js` – JavaScript file.
- `assets/` – Directory for images, fonts, etc.
